# AiYo — Vive contigo.

App de compañía IA. Tu amigo de vida disponible 24/7.

## Deploy en Vercel

1. Sube esta carpeta a GitHub
2. Conecta el repo en vercel.com
3. Deploy automático

## Stack
- Frontend: HTML/CSS/JS puro
- Auth + DB: Supabase
- IA: Claude (Anthropic API)
- Pagos: PayPal Subscriptions
